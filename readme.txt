These files accompany the paper "Recent is more: a negative time-order effect in
non-symbolic numerical judgment" by Van den Berg, Lindskog, Poom, and Winman (2017, JEP:HPP).

Very brief description of the files (more details are found inside the files):

* create_FigXX          : reproduce main figures panels from the paper
* fit_model_expX        : fits the ideal observer model and produces a plot
* get_included_subjects : returns a vector with IDs of subjects included in analyses
* plot_group_fit        : plots a subject-averaged group fit
